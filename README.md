## 🚀 Featured Projects

### 🍱 Yumaste — Food Box E-commerce Platform
> A complete full-stack e-commerce for personalized food boxes, built across 3 repositories.

| Repo | Stack | Description |
|------|-------|-------------|
| [yumaste-backend](https://github.com/SantoFemiano/yumaste-backend) | Java 21 · Spring Boot 4 · MySQL | REST API with JWT auth, orders, warehouse, discounts |
| [yumaste-shop](https://github.com/SantoFemiano/yumaste-shop) | React 19 · TypeScript · TailwindCSS | Client-facing shop with catalog, cart and orders |
| [yumaste-shop-admin](https://github.com/SantoFemiano/yumaste-shop-admin) | Angular 21 · Bootstrap · Chart.js | Admin panel with dashboard and full CRUD |
| [yumaste-db](https://github.com/SantoFemiano/yumaste-db) | MySQL · DDL/DML | Normalized relational schema with 18+ entities |
